### Addition1-java、python中提供的数据结构

#### 1、java

##### 1.1 Queue

- Queue,  Deque。 Queue继承于Collection，且 Deque是Queue的子类
- ArrayDeque实现了Deque接口，LinkedList实现了Deque,List接口



（1）对比：

- ArrayDeque 为数组结构，插入元素不能为null，无法确定数据量时，后期扩容会影响效率
- LinkedList 链表结构，插入元素不能为null，无法确定数据量时，后期扩容会影响效率



（2）用法

- Deque 作为stack：
  - 出栈：poll() 栈空出栈时返回null，pop()则会抛出异常
  - 查看栈顶：peek()，栈空返回null
  - 入栈：push()

- Deque 作为队列
  - 出入队：poll() ,  offer()
  - 插入：
    - 空间不足抛异常：addFirst(),  addLast(),  push():头插， add(): 尾插
    - 空间不足返回false：offerFirst(),  offerLast(),  
  - 抛出：
    - 为空抛异常：
      - 删头：removeFirst(), remove(), pop()
      - 删尾：removeLast(), 
    - 为空时返回null：
      - 删头：pollFirst(),  poll()
      - 删尾：pollLast()

##### 1.2 Stack-不推荐使用

- Stack实现了Vector接口
- Vector虽然线程安全，但效率太低

### 回溯      
回溯的框架     
```Python  
res = []
def backtrack(路径， 选择列表):
    if 满足结束条件：
        res.add(路径)
        return
    for 选择 in 选择列表:
        做选择
        backtrack(路径， 选择列表)
        撤销选择
```
