## 动态规划     
### 区别   
- 与递归、分治没有根本上的区别，关键看有无最优子结构
- 共同点：找到重复子问题
- 不同点：DP中的最优子结构，并且中途可以淘汰次优解

### 关键点：
- 1.最优子结构，optimal = best_of(opt[n - 1], opt[n - 2])
- 2.存储中间状态：opt[i]
- 3.递推公式（状态转移方程 或 DP方程）
  - 斐波那契数列： opt[i] = opt[n - 1] + opt[n - 2]
  - 二维路径(62, 63)： opt[i, j] = opt[i + 1][j] + opt[i][j + 1]，判断a[i, j]是否为空地        

### 解题步骤：
- 1.子问题
- 2.状态定义
- 3.DP方程
